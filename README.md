# drenjongke
This repository contains materials related to the Drenjongke language in Sikkim, India. 

1. **Drenjongke Part-of-Speech Tagged Corpus**

      <ins>Description</ins>
      
      34 Part-of-Speech (PoS) tags, 275 sentences and 1366 tokens.
